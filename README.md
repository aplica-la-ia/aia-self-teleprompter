# AIA-Self-Teleprompter 📱🎬

**AIA-Self-Teleprompter** es una aplicación web progresiva (PWA) diseñada para optimizar la grabación de vídeos. Esta herramienta nace como una **adaptación y extensión para dispositivos móviles** del plugin integrado en **AIA Studio Lite**, permitiendo llevar la potencia del teleprompter segmentado a cualquier tablet o smartphone.

## ✨ Características Principales
- **Origen AIA Studio**: Adaptado para ser el complemento móvil ideal del flujo de trabajo de AIA Studio Lite.
- **Modo Offline**: Una vez instalada como PWA, funciona totalmente sin conexión a internet.
- **Segmentación Inteligente**: Convierte selecciones de texto en tarjetas de lectura individuales de forma táctil.
- **Control Gestual**: Navegación fluida mediante deslizamiento (Swipe) y zoom con dos dedos (Pinch-to-Zoom).
- **Privacidad Local**: Los guiones y ediciones se guardan exclusivamente en el almacenamiento local del dispositivo.

## 🚀 Instalación
1. Accede a la URL de GitHub Pages desde tu tablet o móvil. [🔗](https://pedropablo-dev.github.io/AIA-Self-Teleprompter/)
2. **iOS (Safari)**: Toca el icono "Compartir" y selecciona "Añadir a la pantalla de inicio".
3. **Android (Chrome)**: Toca los tres puntos y selecciona "Instalar aplicación" o "Añadir a la pantalla de inicio".
